Lavoro di tesi: "Un’applicazione desktop per l’esportazione di bibliografia scientifica"

Università della Calabria

Il mio lavoro di tesi è stato quello di creare un’applicazione desktop che permette di aggiungere articoli in formato BibTeX, salvarli all’interno di un database locale e poi di esportarli. Durante la procedura di inserimento, viene data all’utente la possibilità di modificare i dati degli autori/editori e le informazioni dell’articolo. In fase di visualizzazione l’utente può filtrare gli articoli in base al nome, agli autori o al tipo.
È stata realizzata l’applicazione desktop descritta in precedenza utilizzando le seguenti tecnologie: Java, JavaFX, SQLite, Regex, BibTeX

Punti forti di questa applicazione sono: utilizzo offline, interfaccia semplificata, inserimento semplificato, modifica nome e cognome degli autori ed editori.

To-Do App

Università della Calabria

Applicazione Android che fornisce all'utente la possibilità di creare/ modificare/eliminare la lista delle cose da fare. Un' applicazione simile è Google Keep.

Nella mia applicazione è possibile fare il login tramite Google o registrarsi tramite la combinazione email-password. Se l'utente si registra tramite email e password, viene inviata una mail all'indirizzo fornito per confermare la creazione dell'account. Una volta fatto l'accesso , l'applicazione permette di salvare le proprie note sui server di Google, più precisamente è stata usata la piattaforma Firebase per poter sincronizzare un account Google con le note salvate nell'app.
Tecnologie usate: Kotlin, Android Studio, Firebase(e di conseguenza le API di Google).

Le tecnologie usate sono state studiate e approfondite dalle documentazioni ufficiali.
